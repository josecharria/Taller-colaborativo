# PROBLEMA:  Instituto de la Mente Infantil — Uso problemático de Internet

Relacionar la actividad física con el uso problemático de Internet
Resumen

¿Puede predecir el nivel de uso problemático de Internet exhibido por niños y adolescentes, en función de su actividad física? El objetivo de esta competencia es desarrollar un modelo predictivo que analice la actividad física y los datos de aptitud física de los niños para identificar los primeros signos de uso problemático de Internet. La identificación de estos patrones puede ayudar a desencadenar intervenciones para fomentar hábitos digitales más saludables.
